# Product Management CRUD System (SQL)

## About
This is a SQL project focused on practicing CRUD operations.
The project simulates a basic product management system where
products can be created, viewed, updated, and deleted.

---

## Tech Used
- MySQL
- SQL
- VS Code

---

## Project Structure
sql-product-crud-project/
- setup.sql
- crud_queries.sql
- README.md

---

## Database Overview
Database Name: product_db

The database contains a single table `products` which stores
product details like name, category, price, stock, and status.

---

## How to Run
1. Open MySQL
2. Run setup.sql once to create database and insert data
3. Run queries from crud_queries.sql to perform CRUD operations

---

## CRUD Operations Covered
- CREATE: Add new products
- READ: View and filter products
- UPDATE: Modify price and stock
- DELETE: Remove discontinued products
