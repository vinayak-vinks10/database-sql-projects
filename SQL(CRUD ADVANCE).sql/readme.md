# Student Course Management CRUD System (SQL)

## About
This is a large SQL CRUD project that simulates how a college or
training institute manages students, courses, and enrollments.

The project focuses on practicing complete CRUD operations
using multiple related tables.

---

## Tech Used
- MySQL
- SQL
- VS Code

---

## Project Structure
sql-student-course-crud-project/
- setup.sql
- crud_queries.sql
- README.md

---

## Database Overview
Database Name: student_course_db

### Tables
- students
- courses
- enrollments

The enrollments table connects students and courses.

---

## How to Run
1. Open MySQL
2. Run setup.sql once
3. Run queries from crud_queries.sql step by step

---

## CRUD Operations Covered
- CREATE: students, courses, enrollments
- READ: filters, joins, reports
- UPDATE: student and course data
- DELETE: enrollments and inactive records
