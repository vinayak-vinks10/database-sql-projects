# Employee Mini Management System (SQL)

## About
This is a small **beginner-level SQL project**.
I made this project to practice basic database concepts like tables, keys, and simple queries.

The project manages **employees and departments**.

---

## Tech Used
- MySQL
- SQL
- VS Code

---

## Project Structure
sql-employee-mini-project/
- setup.sql
- queries.sql
- README.md

---

## Database Info
Database Name: `company_db`

### Tables Used
- departments – stores department details
- employees – stores employee details

Employees are linked to departments using `dept_id`.

---

## How to Run
1. Open MySQL
2. Run `setup.sql` (only once)
3. Run queries from `queries.sql` for practice

---

## What I Learned
- How to create database and tables
- Primary key and foreign key
- Insert, update, delete data
- Basic select and join queries
