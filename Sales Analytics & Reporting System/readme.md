# Sales Analytics & Reporting System (SQL)

## About
This project focuses on analyzing sales data using SQL.
The main goal is to practice writing analytical and reporting queries
that are commonly used in real business scenarios.

---

## Tech Used
- MySQL
- SQL
- VS Code

---

## Project Structure
sql-sales-analytics-project/
- setup.sql
- queries.sql
- README.md

---

## Database Overview
Database Name: sales_db

The database contains sales data including product, category,
region, date, quantity, and price.

---

## How to Run
1. Open MySQL
2. Run setup.sql once to create database and insert data
3. Execute queries from queries.sql to analyze sales

---

## SQL Concepts Practiced
- Aggregate functions (SUM)
- GROUP BY and HAVING
- CASE statements
- Date functions (MONTH)
- Business-style reporting queries
