# Hotel Management System (SQL)

## About
This is an advanced SQL project that simulates a real-world
hotel management system. The project focuses on using
multiple tables with proper primary and foreign keys.

---

## Tech Used
- MySQL
- SQL
- VS Code

---

## Database Overview
Database Name: hotel_db

### Tables
- guests
- rooms
- bookings
- payments
- staff

All tables are connected using foreign keys.

---

## How to Run
1. Open MySQL
2. Run setup.sql once
3. Execute queries from queries.sql

---

## SQL Concepts Used
- Primary Key & Foreign Key
- Multiple table relationships
- JOIN operations
- Aggregate functions
- Subqueries
- UPDATE and DELETE

---

## Learning Outcome
This project helped me understand how large relational
