# Inventory & Sales Management System (SQL)

## About
This is an intermediate-level SQL project that simulates a basic
inventory and sales system used in shops or warehouses.

The project focuses on managing products, customers, and sales orders
using relational database concepts.

---

## Tech Used
- MySQL
- SQL
- VS Code

---

## Project Structure
sql-inventory-sales-project/
- setup.sql
- queries.sql
- README.md

---

## Database Overview
Database Name: `inventory_db`

### Tables
- products – stores product and stock details
- customers – stores customer information
- orders – stores sales orders

Orders are linked with customers and products using foreign keys.

---

## How to Run
1. Open MySQL
2. Run `setup.sql` once
3. Execute queries from `queries.sql` step by step
