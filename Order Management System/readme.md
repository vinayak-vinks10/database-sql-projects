# Order Management System (SQL CRUD Project)

## About
This is a large SQL CRUD project that simulates a real-world
order management system used in e-commerce applications.

The project focuses on complete CRUD operations using
customers, products, orders, and order items.

---

## Tech Used
- MySQL
- SQL
- VS Code

---

## Project Structure
sql-order-management-crud-project/
- setup.sql
- crud_queries.sql
- README.md

---

## Database Overview
Database Name: order_db

### Tables
- customers
- products
- orders
- order_items

These tables together manage the full order lifecycle.

---

## How to Run
1. Open MySQL
2. Run setup.sql once
3. Execute queries from crud_queries.sql step by step

---

## CRUD Operations Covered
- CREATE: customers, products, orders, order items
- READ: joins, filters, order reports
- UPDATE: order status, stock update
- DELETE: orders and order items
